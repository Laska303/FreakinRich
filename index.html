FreakinRich game

FreakinRich was an academic project developed during my MSc, in which the main goal was to create a game of chance/board game in c# and Windows Forms.

How to play the game

To initiate a new game, the user must click on the first button (red house icon) of the game menu.

From the moment that there are at least two players in game, the dice button is shown to the player that initiated the game. At this point, if that player press the dice button the game starts and no more payers are allowed to join the match.
